# MPLAB Device Blocks for Simulink

![MPLAB Device Blocks for Simulink](MPLAB_Blockset.jpg)

Microchip :registered: official blockset v3.45 supporting up to 300 microcontrollers.
This support package enables code generation of real-time executable for dsPIC, PIC32 and SAM7x from a Simulink model.

The peripheral blocks configure and inserts the peripheral code in the code generated by embedded coder.
A single-tasking and multi-tasking allows efficient execution on chip of advanced multi-rate models.
A custom protocol allows to send data through the UART from the mcu to the computer. The custom picgui GUI allows plotting incoming data in real-time and recording theses for further analysis or injection into model to feed a simulation with real data.

The blockset support the MathWorks features:

- Hardware in the loop (HIL)
- External mode features.
- Code replacement to benefit from the dsPIC internal DSP architecture

No embedded programming knowledge is required. The toolbox transforms any board equipped with a supported mcu into a rapid control prototyping tool and enable a model-based design development scheme.

## Requirements

The support package works with Matlab from release R2010a to R2019a.

- MathWorks
  - Matlab
  - Simulink
  - Embedded Coder
  - Simulink Coder
  - Matlab Coder
- Microchip
  - MPLAB X IDE [download](https://www.microchip.com/MPLABX)
  - xc16 compiler for 16 bits devices [download](https://www.microchip.com/xc16) (dsPIC)
  - xc32 compiler for 32 bits devices [download](https://www.microchip.com/xc32) (PIC32, SAMx7)

## Installation

1. unzip the package
2. execute the .p installer script . Within Matlab, right click on file install.p and select run.

The MCHP_example folder is created in the current folder and examples copied into.
